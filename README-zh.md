## 🎨 关于我
PHPLogger 是一个基于 PHP 的日志记录工具，它可以......
1. 将日志输出到终端或写入到文件，并且可以通过自定义日志的样式、格式和输出位置。
2. 在终端中，各个级别的日志都染上了相应的颜色。此外，你可以自定义每个级别的颜色。
3. 它提供了多种日志级别，方便你根据需要记录不同类型的信息。
